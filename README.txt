== A FAIRE EN PREMIER ==

Lancer le serveur localhost dans /projet et se connecter à accueil.php
=> http://localhost:8080/accueil.php

Se connecter en tant qu'admin (login : admin@cy-tech.fr / mdp : admin) et charger les fichiers élèves afin de créer les comptes.
Pour créer les comptes élèves il faut upload un par un les 3 fichiers choixEtudiantsParcours.


== NAVIGUER SUR LE SITE ==

Connexion en tant qu'étudiant : Lire les login et mdp dans ListeEtudiants.csv

Connexion en tant que responsable admission : login : admission@cy-tech.fr / mdp : admission
	Charger les 3 fichiers : GSI : choixEtudiants1.csv
			       : MI : choixEtudiants3.csv
			       : MF : choixEtudiants2.csv
Connexion en tant qu'admin : Checker les logs et les erreurs remontées.
